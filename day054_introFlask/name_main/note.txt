以下で 初心者にもわかるように整理します。

⸻

1️⃣ __name__ とは？

Pythonの全てのファイル（モジュール）には __name__ という 特別な変数 が自動的に作られます。
	•	スクリプトとして直接実行されたとき → __name__ は "__main__" になる。
	•	他のファイルからインポートされたとき → __name__ にはその モジュール名（ファイル名） が入る。

⸻

2️⃣ if __name__ == "__main__": の意味

if __name__ == "__main__":
    # この中の処理は「直接実行した時だけ」動く

という書き方をすることで、

✅ モジュールとしてインポートされたときには実行されず
✅ 直接ファイルを実行したときだけ動作する

という動作を実現します。

⸻

3️⃣ 具体例で理解する

ファイル: hello.py

def say_hello():
    print("Hello!")

print(f"__name__ in hello.py: {__name__}")

if __name__ == "__main__":
    say_hello()

ファイル: main.py

import hello


⸻

実行パターン

① python hello.py で直接実行した場合
	•	__name__ は "__main__" になる。
	•	出力:

__name__ in hello.py: __main__
Hello!



② python main.py で実行した場合
	•	hello はインポートされるだけで __name__ は "hello" になる。
	•	if __name__ == "__main__": の中身は実行されない。
	•	出力:

__name__ in hello.py: hello

（Hello! は表示されない）

⸻

4️⃣ 何のために使うの？

✅ モジュールをテスト実行する時だけ動く処理を作りたいとき
✅ 他のファイルから呼び出したときには実行されないようにしたいとき

に使います。

⸻

✅ 一行まとめ

if __name__ == "__main__": は「このファイルを直接実行したときだけ動かす処理」を書くためのPythonの習慣的な書き方です。
